# Subdomains

- Core: O que dá dinheiro
- Supporting: Da suporte para o core funcionar
- Generic: Necessário, mas não são importantes

## Exemples

### Core

- Devem ser desacopladas uma da outra, não podem depender de funções diretamente entre si

- Compra
- Catálogo
- Pagamento
- Entrega
- Faturamento

### Supporting

- Estoque

### Generic

- Notificação
- Promoções
- Chat
